2024的核心: **行动力、专注、独立**.

<!--more-->

## 生活

|          | 要求                   | 原因                                                       |
| -------- | ---------------------- | ---------------------------------------------------------- |
| 居住环境 | 独居                   | 我不想被别人打扰(**专注**), 尤其是自己很累/心情不好的时候. |
| 办公环境 | 高效率、人体工学、温馨 | 让自己在家工作的更舒服.                                    |
| 户外运动 | 挑战自我               | 提高有氧能力、磨练意志力                                   |
| 在家运动 | 简单、高效             | 节省时间                                                   |
| 健身房   | 坚持                   | 提高力量                                                   |
| 做视频   | 用心做有价值的内容     | 费曼学习法、提高表达能力                                   |


## 财富

|          | 要求                           | 原因           |
| -------- | ------------------------------ | -------------- |
| 投资     | 定投纳斯达克、跟踪股票         | 积累财富       |
| 年终奖   | 保证健康的前提下努力争取       | 年终奖不少     |
| 主业收入 | 稳定的积累                     | 原始财富       |
| 副业收入 | 自己当老板、不能单纯的出卖时间 | 风险与机会并存 |
